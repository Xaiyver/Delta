Description
1) A webpage where the main menu shows "modules" of math to enable/disable, e.g., basic arithmetic, definite integrals, linear algebra determinants, differential and integral calculus
2) On the main page, a button to start the game
3) The main page generates randomized single expressions (ideally via a lightweight webengine LaTeX renderer like KaTeX) from the enabled modules that evaluate to a nice integer from -100 to 100
4) After 2 minutes, the run ends and the score is stored in a lightweight database

Technical implementation
1) Code files need to be separated (JavaScript, CSS, HTML as distinct files that are linked together)
2) Each file should do only one thing or one class of things (a single JavaScript file should not be responsible for the entire website)

Basic arithmetic module
1) Addition, subtraction, multiplication, division, exponentiation, logarithms, reasonably simple for immediate mental calculation

Visuals
1) The page should follow strictly a standard solarized light colour scheme for readability
2) Everything should be satisfying and minimalistic, only minor animations, nothing jarring
